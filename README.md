# Fakedex (Next.js + Tailwind) - JavaScript

Projeto gerado automaticamente para você editar manualmente o arquivo `/data/fakemon.json`.
Para rodar:

1. `npm install`
2. `npm run dev`
3. Abra http://localhost:3000

Estrutura principal:
- components/: componentes React
- pages/: rotas Next.js (index + fakemon/[id])
- public/images/: imagens de exemplo (SVG)
- data/fakemon.json: arquivo JSON editável manualmente
